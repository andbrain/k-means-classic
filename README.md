# k-means-classic
Classic k-means using euclidean distance in C++

## Installation
- Cmake (3.5 or >)
> cmake . <BR/>
> make

## Usage
> ./kmeans.out \<K\> \<INPUT\> \<OUTPUT\> \<TYPE\> \<FEATURES\> <BR/>

- K => Number of clusters
- INPUT => File path of input collection
- OUTPUT => File path for clusters file
- TYPE =>
	- 0 -> for distance
	- 1 -> for similarity
